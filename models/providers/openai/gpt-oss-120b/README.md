### API使用例
```python
import openai

client = openai.OpenAI(api_key="your-api-key")

response = client.chat.completions.create(
    model="gpt-oss-120b",
    messages=[
        {"role": "system", "content": "You are a helpful Japanese customer support assistant."},
        {"role": "user", "content": "商品の返品について教えてください"}

    max_tokens=500,
    temperature=0.3
)
```
